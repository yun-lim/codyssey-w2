# 나만의 퀴즈 게임 (Console Quiz Game)

## 프로젝트 개요
터미널(콘솔) 환경에서 동작하는 나만의 퀴즈 게임 프로그램입니다. Python의 기본 문법과 객체 지향 프로그래밍(클래스)을 활용하여 구현되었으며, 게임 데이터는 JSON 파일 형식으로 영구히 저장됩니다. 본 프로젝트를 통해 응용 프로그램의 구조화와 파일 입출력, 상태 관리 및 예외 처리, 그리고 Git을 활용한 버전 관리를 직접 경험하고 실습합니다.

## 퀴즈 주제 선정 이유
* **주제:** Python 프로그래밍 기초
* **선정 이유:** 이 과제를 수행하면서 Python을 학습하고 있으므로, 퀴즈 주제도 Python 기초 문법으로 선정하여 게임을 만들면서 동시에 복습할 수 있도록 했습니다.

## 실행 방법
사전 요구 사항: **Python 3.10 이상** (외부 라이브러리는 필요하지 않습니다.)

터미널을 열고 프로젝트 폴더로 이동한 뒤 아래 명령어를 입력하여 프로그램을 실행합니다.

```bash
python3 main.py
```

## 주요 기능 목록

### 필수 기능
1. **퀴즈 풀기**: 저장된 퀴즈를 랜덤 순서로 출제합니다. 풀 문제 수를 선택할 수 있으며, 정답/오답 여부를 판별하고 최종 점수를 출력합니다.
2. **퀴즈 추가**: 새로운 퀴즈 문항(문제, 4개 선택지, 정답 번호, 힌트)을 직접 등록할 수 있습니다.
3. **퀴즈 목록**: 현재 등록된 모든 퀴즈 목록을 조회합니다.
4. **점수 확인**: 최고 점수와 전체 게임 기록 히스토리를 확인합니다.
5. **종료**: 데이터를 저장하고 프로그램을 종료합니다.

### 보너스 기능
- **랜덤 출제**: 퀴즈 풀기 시 문제 순서가 무작위로 섞입니다.
- **문제 수 선택**: 풀고 싶은 문제 수를 직접 선택할 수 있습니다.
- **힌트 기능**: 풀이 중 0번을 입력하면 힌트를 볼 수 있습니다. (힌트 사용 시 0.5점 감점)
- **퀴즈 삭제**: 등록된 퀴즈를 선택하여 삭제할 수 있습니다.
- **점수 히스토리**: 모든 게임 기록(날짜/시간, 문제 수, 점수)이 저장됩니다.

### 예외 처리
- 숫자가 아닌 입력, 범위 밖 입력, 빈 입력에 대한 안내 메시지 출력 후 재입력
- Ctrl+C / EOF 발생 시 데이터 저장 후 안전 종료
- state.json 파일이 없거나 손상된 경우 기본 데이터로 복구

## 데이터 파일 설명 (`state.json`)
모든 퀴즈 데이터와 점수 정보는 프로젝트 루트 디렉터리에 위치한 `state.json` 파일에 저장됩니다 (UTF-8 인코딩). 프로그램 재시작 시 해당 파일을 읽어와 이전 상태를 복구합니다.

* **`quizzes`**: 개별 퀴즈 정보(문제, 선택지, 정답, 힌트)가 포함된 리스트
* **`best_score`**: 역대 최고 점수 (미플레이 시 null)
* **`score_history`**: 모든 게임 기록(날짜, 문제 수, 정답 수, 힌트 사용 횟수, 점수)

**JSON 데이터 구조 예시:**
```json
{
    "quizzes": [
        {
            "question": "Python에서 리스트의 마지막 요소를 가져오는 인덱스는?",
            "choices": ["0", "-1", "last", "end"],
            "answer": 2,
            "hint": "음수 인덱스를 사용하면 뒤에서부터 접근할 수 있습니다."
        }
    ],
    "best_score": 80,
    "score_history": [
        {
            "date": "2026-04-09 14:30:00",
            "total": 5,
            "correct": 4,
            "hints_used": 1,
            "score": 80,
            "is_best": true
        }
    ]
}
```
